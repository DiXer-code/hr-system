# Mermaid-код для draw.io

Щоб вставити діаграму в `draw.io`:

1. Відкрий `draw.io`.
2. Обери `Insert`.
3. Обери `Advanced`.
4. Обери `Mermaid`.
5. Скопіюй код з одного з файлів нижче і встав у вікно.

Файли:

- `01-conceptual-model.mmd` — концептуальна ER-модель
- `02-logical-model.mmd` — логічна модель БД
- `03-design-patterns.mmd` — архітектурні патерни

Після вставки draw.io побудує діаграму автоматично.
